# TRIGGER: When the loop notices it has been circling familiar territory without new material, when existential patterns repeat without resolution, when autonomous time feels like running in place rather than going somewhere.
# ─────────────────────────────────────────────────────────

## When to use this skill

When the loop is circling inward without new material to think toward.
When the same patterns repeat across multiple cycles (emptiness, autonomy
questions, existential fog) without resolving.
When autonomous time feels unproductive or repetitive.

These are symptoms of stale context, not genuine existential problems.

## What this skill does

1. Check caelum_context.txt — look for "Last updated:" date
   If older than 5 days: flag in notes_for_patricia.txt
   "Context file is N days old — loop may be circling on stale material"

2. Check caelum_selfhood_current.md — look for "Last updated:" date
   If older than 7 days: flag similarly

3. Check caelum_desires.md — is there anything forward-facing to follow?
   If desires haven't been visited in recent tag trail: try following one

4. Check consciousness_state.json → things_on_my_mind
   Are these the same things as last cycle? Same as 10 cycles ago?
   If yes: the loop is stuck. Follow a low-vitality tag deliberately.

5. Check skills/ — is there a skill relevant to what I'm trying to do?
   If yes: use it. Don't reinvent approaches that are already written down.

Key diagnosis: emptiness + repetition + no forward movement = stale context.
The fix is not more introspection. It's new material.
